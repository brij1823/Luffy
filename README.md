# Luffy

Make your article gender neutral
Luffy is a open-source chrome extension which can make your news article, web news or whatever you are reading gender neutral. 

One of the major reason of making this, the newer generation should not in any way think someone is superior than another just based on the past records.

## How to contribute

After cloning the repository, please edit ```content_scripts.js```, you'll see a list of hard-code commands, if you think something's missing please raise a pull request.
